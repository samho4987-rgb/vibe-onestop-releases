# 유튜브 스크립트 skill (vibe-onestop 4단계 ④)

입력 제한: selling_points(검수완료)·location_points·market·register·deal·features. contacts 제외.
표시광고 규정은 영상에도 적용 — 화면 자막에 소재지(동)·면적(대장)·가격·중개사무소 명칭 고지.

## 숏폼 60초 (세로 9:16)

| 구간 | 초 | 내용 규칙 |
|---|---|---|
| 훅 | 0~5 | 질문 or 반전 숫자 하나 ("하단역 3분, 실투자 1억이면?") — 건물 외관 컷 |
| 위치 | 5~15 | 역·상권 팩트 2개, 지도 줌 화면 |
| 내부 | 15~35 | 구조·면적·특징 3컷 (복층 등), 자막은 대장 기준 면적 |
| 조건 | 35~50 | 가격·임차승계·수익 실계산 자막 ("연 임대료 1,440만 = 매매가 대비 8%") |
| CTA | 50~60 | "자세한 조건은 고정댓글·채널 프로필" — 전화번호 발화·자막 금지 |

- 발화 속도 기준 전체 140~160자/10초, 총 850자 이내
- 컷마다 [화면지시]와 (자막)을 분리 표기

## 롱폼 (3~5분)

1. 인트로 (30초): 지역 상권 맥락 — 왜 이 동네인가
2. 외부·입지 (1분): location_points 순회, 도보 실측 언급
3. 내부 투어 (1~2분): 공간별 서술, 추천업종 시뮬레이션 ("여기에 테이크아웃 카페를 넣으면…")
4. 조건·수익 분석 (1분): verified 실계산, 융자·실투자 구조 설명. claim은 "매도인 제시"로 구분
5. 유의사항 (30초): 솔직 고지 1~2개 (신뢰 장치)
6. 아웃트로: 사무소 소개 + 구독 유도

## 공통 규칙

- 수익 "보장" 표현 금지, 미래 시세 예측 단정 금지
- 세입자 있는 매물은 내부 촬영 동의 확인 자막 처리
- 산출물 형식: 씬 번호 / [화면] / 대사 / (자막) 4열 구성
