# SNS skill (vibe-onestop 4단계 ⑤)

입력 제한: selling_points(검수완료)·location_points·register·deal·features. contacts 제외.

## 인스타 문구

- 구조: 훅 1줄 (이모지 1~2개) → 핵심 팩트 3줄 (역·면적·가격) → 수익/조건 1줄 (실계산만) → CTA ("자세한 내용은 프로필 링크")
- 본문 300자 이내, 줄바꿈으로 호흡
- 해시태그: 지역 3개(#하단동 #하단역 #사하구상가) + 상품 3개(#상가매매 #코너상가 #수익형부동산) + 브랜드 1개 — 총 7±2개
- 전화번호·개인정보 금지. 표시광고 의무 문구는 카드 마지막 장에 (아래)

## Canva 카드뉴스 지시서 (5장 기본)

각 장을 `{페이지, 목적, 헤드카피, 서브카피, 비주얼 지시}` 로 기술.

| 장 | 목적 | 규칙 |
|---|---|---|
| 1 | 훅 | 숫자 하나 크게 ("하단역 3분"). 외관 사진 배경 |
| 2 | 입지 | 지도 + location_points 3개 아이콘 표기 |
| 3 | 물건 | 내부 사진 + 대장 기준 스펙 표 (용도·전용㎡·층) |
| 4 | 조건 | 가격·임차승계·실계산 수익. "보장" 표현 금지 |
| 5 | 의무기재+CTA | 소재지(동)·면적·가격·관리비·주차·중개사무소 명칭·등록번호 (office.json) |

- 톤: 브랜드 컬러 `#3fbf7f` 포인트, 어두운 배경 + 흰 텍스트
- 이미지 산출은 `04_images.json`으로 저장 (realty-pipeline save_package 규격):
  `[{"page": 1, "headline": "...", "sub": "...", "visual": "...", "size": "1080x1080"}]`
