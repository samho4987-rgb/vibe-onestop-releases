# vibe-onestop 플러그인 배포소

부동산 매물접수 업무자동화 파이프라인 — Claude 플러그인.
접수문자 파싱부터 공부·시세 검증, 홍보 콘텐츠(블로그·유튜브·SNS) 생성까지 한 번에.

## 설치 방법 (둘 중 하나)

### 방법 1. 마켓플레이스 등록 — 권장 (자동 업데이트)

1. Claude 데스크톱 앱을 열고, 왼쪽 사이드바 맨 아래 **사용자 지정**을 클릭
2. 설정 창이 열리면 왼쪽 목록 아래쪽 **사용자 지정** 항목에서 **플러그인** 클릭
3. 오른쪽 위 **추가 ▾** 버튼 → **마켓플레이스 추가**
4. **저장소에서 추가** 선택
5. URL 칸에 아래 주소를 **붙여넣고** **동기화** 클릭

   ```
   samho4987-rgb/vibe-onestop-releases
   ```

   > 목록에는 본인 GitHub 저장소만 표시되므로, 위 주소는 직접 붙여넣어야 합니다.

6. 목록에 나타난 **Vibe onestop** 설치

새 버전이 나오면 마켓플레이스에서 업데이트만 누르면 됩니다.

### 방법 2. 파일로 설치

1. [Releases](../../releases)에서 최신 `vibe-onestop-1.4.1.plugin` 다운로드
2. Claude 데스크톱 앱 왼쪽 사이드바 맨 아래 **사용자 지정** 클릭
3. 설정 창에서 **플러그인** 클릭
4. 오른쪽 위 **추가 ▾** → **플러그인 업로드**
5. 내려받은 `.plugin` 파일 선택

> `.plugin` 파일은 더블클릭으로 설치되지 않습니다. 반드시 Claude 앱 안에서 등록하세요.

## 설치 후 반드시 할 것 — 내 컴퓨터에서 실행

동봉된 MCP 서버는 **Windows 실행파일**입니다. Cowork 작업을 클라우드에서 실행하면
(기본값) 서버가 뜨지 않아 매물장·시세조회 도구를 찾지 못합니다.

1. **첫 실행 시 Windows 기능 활성화** 안내가 뜨면 진행하고 **재부팅** 합니다
   - 켜지는 기능: **가상 머신 플랫폼(Virtual Machine Platform)** — 최초 1회
   - Windows **Home 에디션에서도 지원**됩니다 (Hyper-V·Windows 샌드박스와 달리 Pro가 필요 없음)
2. 작업을 시작할 때 실행 위치를 **내 컴퓨터**로 선택합니다
   - **매 작업마다 확인하세요.** 기본값이 클라우드입니다
3. 확인 방법: 오른쪽 패널에 **작업 폴더**가 표시되면 내 컴퓨터에서 실행 중입니다
   - 이때 도구 목록에 `vibe-onestop:vibe-ledger` 등이 커넥터로 잡힙니다

동작 확인용 (읽기 전용):

> vibe-ledger get_stats 실행해줘

매물장 통계와 DB 경로(`C:\Users\<사용자>\.greencore\listings.db`)가 나오면 정상입니다.
처음에는 0건이 정상이며, 접수문자를 넣으면 채워집니다.

## 설치 후 설정

API 키는 설치 후 대화에서 등록합니다 (한 번만).

| 키 | 발급처 | 등록 방법 |
|---|---|---|
| 카카오 REST API | developers.kakao.com | "카카오 키 등록해줘" → `set_api_key` |
| 공공데이터포털 | data.go.kr (**Decoding** 키) | "공공데이터 키 등록해줘" |
| 네이버 검색 API | developers.naver.com | "네이버 키 등록해줘" |

## 구성 (v1.3)

| 구성요소 | 내용 |
|---|---|
| 스킬 6종 | pipeline · intake-parsing · ad-copy · blog-post · youtube-script · sns-card |
| MCP 서버 4종 (동봉) | vibe-ledger(매물장) · vibe-geo(입지) · vibe-pubdata(공부·시세 검증) · vibe-news(뉴스 글감) |

- **지원 OS**: 동봉 MCP 서버 실행파일은 **Windows x64 전용**입니다. macOS에서는 스킬만 동작하며 MCP 서버는 Python으로 별도 등록이 필요합니다.
- **요금제**: 플러그인은 유료 플랜(Pro/Max/Team/Enterprise)에서 사용할 수 있습니다.
- 매물 데이터는 사용자 PC에만 저장됩니다 (`~/.greencore/listings.db`). 외부로 전송하지 않습니다.

## 사용법

플러그인 설치 후 Claude에게:

> 접수문자 [원문 붙여넣기] 처리해줘

pipeline 스킬이 접수 → 검증 → 콘텐츠 생성 표준 시퀀스를 실행합니다.
임대인 광고 동의 확인 · 홍보문구 최종 검수 · 광고 등록 실행은 **항상 사람이** 합니다.
